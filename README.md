Personal website for [http://kamaths.github.io/](http://kamaths.github.io/) built with [Jekyll](http://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

## Requirements

```sh
gem install github pages
```

## Usage

```sh
jekyll serve
```
